M_UI_GaugeFlying rebuild notes
================================

Goal
----
Rebuild a UI gauge material similar to M_UI_GaugeFlying using these source textures:

- Texture2D'/AION2/Content/UI/Resource/Texture/UIFX/UT_Flare_044.UT_Flare_044'
- Texture2D'/AION2/Content/UI/Resource/Texture/UIFX/UT_Gradient_006.UT_Gradient_006'
- Texture2D'/AION2/Content/UI/Resource/Texture/UIFX/UT_Gradient_016.UT_Gradient_016'
- Texture2D'/AION2/Content/UI/Resource/Texture/UIFX/UT_HUD_FPGauge_FG.UT_HUD_FPGauge_FG'
- Texture2D'/AION2/Content/UI/Resource/Texture/UIFX/UT_Noise_010.UT_Noise_010'

Recommended material settings
-----------------------------

- Material Domain: User Interface
- Blend Mode: Translucent or Additive
- Shading Model: Unlit
- Two Sided: false

Parameters
----------

- HitValue: Scalar, default 0.75, range 0..1
- GlowPower: Scalar, default 3.0
- EdgeGlowWidth: Scalar, default 0.035
- FlowSpeed: Scalar, default 0.35
- NoiseStrength: Scalar, default 0.14
- HueShift: Scalar, default 0.0
- BaseTint: Vector, default (R=0.05, G=0.85, B=1.0, A=1.0)
- HotTint: Vector, default (R=1.0, G=0.85, B=0.18, A=1.0)

Texture parameters
------------------

- T_Flare: UT_Flare_044
- T_GradientA: UT_Gradient_006
- T_GradientB: UT_Gradient_016
- T_GaugeFG: UT_HUD_FPGauge_FG
- T_Noise: UT_Noise_010

Custom node version
-------------------

Create one Custom node:

- Description: GaugeFlying_HitValue_Glow
- Output Type: CMOT Float 4
- Inputs:
  - UV: float2
  - HitValue: float
  - GlowPower: float
  - EdgeGlowWidth: float
  - FlowSpeed: float
  - NoiseStrength: float
  - HueShift: float
  - BaseTint: float3
  - HotTint: float3
  - TimeValue: float
  - T_Flare: Texture2D
  - T_FlareSampler: SamplerState
  - T_GradientA: Texture2D
  - T_GradientASampler: SamplerState
  - T_GradientB: Texture2D
  - T_GradientBSampler: SamplerState
  - T_GaugeFG: Texture2D
  - T_GaugeFGSampler: SamplerState
  - T_Noise: Texture2D
  - T_NoiseSampler: SamplerState

Connect the Custom node RGB to Final Color, and A to Opacity.

Paste this code into the Custom node:

```hlsl
float2 uv = UV;
float fill = saturate(HitValue);

float noise = T_Noise.Sample(T_NoiseSampler, uv * float2(2.0, 1.0) + float2(TimeValue * FlowSpeed, 0.0)).r;
float flow = T_GradientA.Sample(T_GradientASampler, uv + float2(TimeValue * FlowSpeed + noise * NoiseStrength, 0.0)).r;
float band = T_GradientB.Sample(T_GradientBSampler, float2(frac(uv.x + TimeValue * FlowSpeed * 0.65), uv.y)).r;
float gauge = T_GaugeFG.Sample(T_GaugeFGSampler, uv).r;
float flare = T_Flare.Sample(T_FlareSampler, float2(frac(uv.x - TimeValue * FlowSpeed * 1.35), uv.y)).r;

float filledMask = step(uv.x, fill);
float edge = 1.0 - smoothstep(0.0, max(EdgeGlowWidth, 0.0001), abs(uv.x - fill));
float railFade = smoothstep(0.0, 0.08, uv.y) * (1.0 - smoothstep(0.92, 1.0, uv.y));

float huePhase = frac(HueShift + TimeValue * 0.08 + flow * 0.18);
float3 dynamicTint = lerp(BaseTint, HotTint, saturate(huePhase * 1.7));
dynamicTint = lerp(dynamicTint, BaseTint.brg, smoothstep(0.55, 1.0, huePhase));

float body = filledMask * gauge * railFade;
float glow = (body * (0.35 + flow * 0.65) + edge * 1.7 + flare * band * filledMask) * GlowPower;
float3 color = dynamicTint * body + dynamicTint * glow;
float alpha = saturate(body + edge * 0.85 + flare * filledMask * 0.55);

return float4(color, alpha);
```

Blueprint-style graph recipe
----------------------------

If you prefer regular material nodes instead of a Custom node, build the same graph like this:

1. TextureCoordinate -> all texture UV inputs.
2. Time * FlowSpeed -> add to UV.x for T_GradientA, T_GradientB, T_Flare, and T_Noise.
3. T_Noise.r * NoiseStrength -> extra offset for T_GradientA UV.x.
4. Step mask equivalent:
   - If( A=TextureCoordinate.r, B=HitValue, AGreaterThanB=0, AEqualsB=1, ALessThanB=1 )
5. Edge glow:
   - Abs(TextureCoordinate.r - HitValue)
   - SmoothStep(0, EdgeGlowWidth, result)
   - OneMinus
6. Body:
   - FilledMask * T_GaugeFG.r * vertical rail fade
7. Glow:
   - Body * lerp(0.35, 1.0, T_GradientA.r)
   - plus EdgeGlow * 1.7
   - plus T_Flare.r * T_GradientB.r * FilledMask
   - multiply by GlowPower
8. Color:
   - Lerp(BaseTint, HotTint, animated HueShift/Time/Gradient value)
   - multiply by Body + Glow
9. Output:
   - RGB -> Final Color
   - Alpha -> Opacity

Runtime control
---------------

From UMG or C++/Blueprint, drive the gauge by setting:

- HitValue: current / max, clamped 0..1
- GlowPower: higher when taking damage or on highlight
- HueShift or BaseTint/HotTint: change color over time or by state
